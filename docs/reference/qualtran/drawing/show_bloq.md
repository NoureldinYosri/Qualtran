<h2 class="add-link">Args</h2>

`bloq`<a id="bloq"></a>
: The bloq to show

`type`<a id="type"></a>
: Either 'graph' or 'musical_score'. By default, display a directed acyclic
  graph of the bloq connectivity. Otherwise, draw a musical score diagram.
