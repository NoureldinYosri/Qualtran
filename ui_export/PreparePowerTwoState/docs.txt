Prepares the uniform superposition over $|r\rangle$ given by Eq. 69 in the reference.

This prepares the state

$$
    2^{(-n_p -1)/2} \sum_r=0^{n_p-2} 2^{r/2} |r\rangle
$$

in one-hot unary.

#### Parameters
 - `bitsize`: the number of bits $n_p$ for the $|r\rangle$ register. 

#### Registers
 - `r`: The register we want to prepare the state over. 

#### References
 - [Fault-Tolerant Quantum Simulations of Chemistry in First Quantization](https://arxiv.org/abs/2105.12767). Eq 67-69, pg 19-20
